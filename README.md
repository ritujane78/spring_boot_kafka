# Kafka Stream (In Progress)

This Spring Boot project, built using Gradle, demonstrates how to use Kafka Streams to process and manipulate data from Kafka topics.

The data consumed from the Kafka topics is produced and stored through REST APIs defined in the `kafka-stream-order` project. 

This project focuses on stream processing operations such as filtering, mapping, branching, aggregation, transformation, and topic-to-topic data flow using Kafka Streams.

With docker, build and run a docker container for this project.
